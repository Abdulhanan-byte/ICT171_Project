ICT171 Cloud Server Project

Hosted on Microsoft Azure Virtual Machine

Project Overview

This project is a simple cloud-hosted website deployed on a Microsoft Azure Virtual Machine. The website demonstrates how Infrastructure as a Service can be used to create, configure, and publish a web server online.

Server Environment

Cloud Provider

Microsoft Azure

Server Type

Linux Virtual Machine

Web Server

Nginx

Access Method

Public IP Address

Completed Tasks

  • Connected to the Azure server using SSH.
  • Removed the previous website files from the web directory.
  • Created a new HTML website for the project.
  • Configured the website inside the Nginx web root folder.
  • Published the website using the Azure public IP address.

Learning Outcome

This project helped me understand Linux server access, Nginx web hosting, cloud infrastructure, file management, and the practical use of Azure IaaS for hosting a website.
