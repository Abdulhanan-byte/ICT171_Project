Project Description

My cloud project is a simple multi-page blog website running on Ubuntu and Nginx on a Microsoft Azure virtual machine. This project demonstrates how an Infrastructure-as-a-Service environment can be used to host and manage a working web server.

Purpose

The purpose of this project is to practise SSH access, Linux server management, Nginx configuration, website deployment, firewall setup, and basic cloud infrastructure management.
