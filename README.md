## Project Overview

This project is a simple cloud-hosted website deployed on a Microsoft Azure Virtual Machine using Infrastructure as a Service. The website demonstrates Linux server access, Nginx web hosting, public IP deployment, and basic cloud server management.

## Live Website

http://20.92.248.197/

## Server Environment

- Cloud Provider: Microsoft Azure
- Server Type: Linux Virtual Machine
- Web Server: Nginx
- Hosting Method: Infrastructure as a Service
- Access Method: Public IP Address
- Web Root Directory: /var/www/html

## Project Tasks Completed

- Connected to the Azure Virtual Machine using SSH.
- Removed the previous website files from the Nginx web directory.
- Created a new static HTML website.
- Deployed the website into /var/www/html.
- Restarted Nginx to publish the updated website.
- Prepared the project files for GitHub version control.

This project helped me understand how an Azure virtual machine can be used to host a website using Linux and Nginx. It also demonstrates the practical use of Infrastructure as a Service for deploying and managing a simple web server.
